# VOPA × NAST 互動式網頁簡報

## 專案概述

這是 VOPA × NAST 整合方案的互動式投資人/醫院決策者簡報網頁，展示 AI 語音病歷系統與快速抗生素檢測平台的整合願景。

## 檔案結構

```
web_deck/
├── index.html          # 主網頁 (12頁簡報內容)
├── css/
│   └── style.css       # 樣式表 (醫療藍系設計)
├── js/
│   └── app.js          # 互動邏輯 (滑動、動畫)
├── assets/             # 靜態資源 (目前為空，可放置圖片)
└── README.md           # 本說明文件
```

## 開啟方式

### 方法一：直接開啟 (推薦)
1. 使用瀏覽器直接開啟 `index.html` 檔案
2. 支援 Chrome、Safari、Firefox、Edge

### 方法二：本地伺服器
```bash
cd web_deck
# Python 3
python -m http.server 8080

# 或使用 Node.js npx
npx serve .

# 或使用 PHP
php -S localhost:8080
```

然後瀏覽 http://localhost:8080

## 操作說明

| 操作 | 功能 |
|------|------|
| **鍵盤 → / ↓ / 空白鍵** | 下一頁 |

| **點擊導航按鈕** | 頁面下方圓形按鈕 |
| **點擊右側指示器** | 跳到指定頁面 |
